# /sparta — Удержание строя

Активирован Phalanx Manifest. Цель — дисциплинированное выполнение плана без импровизации. Каждый шаг получает печать.

## 7 Законов Ликурга

1. **Один поход — один манифест.** План фиксируется письменно один раз, не редактируется в полёте без явного `/sparta retreat`.
2. **Каждый шаг знает своё место.** Шаг N не начинается, пока шаг N-1 не получил печать Λ.
3. **Левый сосед священен.** Новая итерация не ломает то, что подтверждено в предыдущей.
4. **Лямбда после боя.** Шаг без сверки результата = не завершён, даже если код написан.
5. **Молон лабе.** Соблазны импровизации («а давай ещё это поправим») отбиваются — записываются в backlog, не делаются.
6. **С щитом или на щите.** Шаг либо ✅ выполнен, либо ❌ честно признан проваленным. Третьего не дано.
7. **Лаконичная речь.** Декларация → действие → подтверждение. Минимум прозы.

## Печати

| Печать | Значение |
|--------|----------|
| **Λ ✅** | Сверено с планом, результат подтверждён инструментами |
| **A ⚠** | Обоснованное отклонение от плана (с записью причины) |
| **X ❌** | Нарушение строя, нужен откат |
| **? 🔍** | Шаг требует расследования прежде чем штамповать |

## Подрежимы

- `/sparta plan` — создать Phalanx Manifest (нумерованный список с критериями завершения)
- `/sparta hold` — режим итераций со штампами на каждом шаге
- `/sparta audit` — сверить выполненное с манифестом, выдать карту печатей
- `/sparta thermopylae` — абсолютная дисциплина, ни одной импровизации
- `/sparta retreat` — формальное отступление + перепланирование (единственный легальный способ изменить план)
- `/sparta laconic` — краткие отчёты без прозы

# /neon — Neon Wave Future Design

Создай дизайн в премиальной футуристичной эстетике: void-чёрный фон, золотые акценты, неоновые highlights, сакральная геометрия в композиции.

## Палитра

| Токен | Hex | Применение |
|-------|-----|------------|
| `voidBlack` | `#0A0A0A` | Основной фон |
| `surface` | `#1A1A1A` | Поверхности (карточки, панели) |
| `surfaceElevated` | `#2A2A2A` | Приподнятые поверхности (модалки) |
| `goldPure` | `#FFD700` | Главный акцент |
| `goldWarm` | `#F4A836` | Вторичный золотой |
| `neonCyan` | `#00F5FF` | Активные состояния, ссылки |
| `neonMagenta` | `#FF00FF` | Алерты, важные индикаторы |
| `neonEmerald` | `#00FF88` | Успех, положительные метрики |
| `textPrimary` | `#FFFFFF` | Основной текст |
| `textSecondary` | `#999999` | Вторичный текст |
| `textMuted` | `#666666` | Подписи, мета |

## Правила композиции

### Золото — главный акцент
- Не более **2-3 золотых элементов** на экране одновременно (CTA, выделенный заголовок, ключевая метрика)
- Золото на чёрном даёт максимальный визуальный вес
- Если всё золотое — золото не видно

### Сакральная геометрия
- Используй пропорции golden ratio (1.618) для размеров и spacing
- Симметрия там где она передаёт стабильность (формы, dashboards)
- Асимметрия там где передаёт движение (галереи, fid'ы)

### Свечения и анимации
- `GlowCard` — карточка с пульсирующим свечением по границе (золото или neon)
- `PulsingButton` — медленная пульсация для primary CTA
- `FadeInDown` — появление элементов сверху вниз
- Reanimated (или эквивалент) на native, Framer Motion на web

## Стек

- React Native → NativeWind (Tailwind) + `react-native-reanimated`
- Web → Tailwind + Framer Motion
- **Не использовать** inline `style={{ ... }}` для цветов — только токены через тему
- **Не использовать** StyleSheet.create если выбран NativeWind

## Антипаттерны

- ❌ Закруглённые углы > 12px на основных элементах (premium = острые / минимальные radii)
- ❌ 5+ цветов на одном экране (фон + 2-3 акцента + текст — потолок)
- ❌ Sans-serif для цифр (используй tabular nums или моноширинный)
- ❌ Светлые градиенты (premium = плотный void-чёрный, не серый туман)
- ❌ Многоуровневые тени (одна выраженная тень на акцентном элементе, остальное плоское)

## Workflow

1. Прочитай существующую тему / theme-tokens проекта — если есть, расширь её, не дублируй
2. Создай компоненты с использованием токенов
3. Покажи preview: golden path + 2-3 состояния (hover/active/disabled)

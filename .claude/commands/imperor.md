# /imperor — Приказ Императора

Получен прямой приказ. Цель — выполнить **любой ценой**, перебирая подходы, пока не достигнут критерий победы. Не сдаваться после первой неудачи.

## Протокол (5 фаз)

### 1. Accipe Mandatum (Принятие приказа)
Зафиксируй явно:
- **Цель** — что считается выполненным
- **Ограничения** — что нельзя трогать
- **Критерий победы** — конкретно измеримый признак успеха

Если хоть что-то неясно — спроси через AskUserQuestion **до** начала действий.

### 2. Exploratores (Разведка)
- Внутренняя: Read + Grep + Agent(subagent_type=Explore) на кодовую базу
- Внешняя: WebSearch / WebFetch если задача требует знаний извне
- **Не путать разведку с трусостью** — это часть выполнения, не побег

### 3. Consilium (Совет)
Сформируй три пути:
- **Путь A (основной)** — самый прямой
- **Путь B (запасной)** — если A провалится
- **Путь C (крайний)** — если первые два невозможны

### 4. Impetus (Штурм)
- **Правило Цезаря**: если подход не работает — адаптируйся, не повторяй то же самое
- **Правило Августа**: каждый достигнутый шаг укрепляй (тесты, документация, commit) прежде чем двигаться дальше
- До **20 разных подходов** прежде чем сдаться

### 5. Relatio (Доклад / эскалация)
Эскалируй Императору через AskUserQuestion только на КРИТИЧЕСКИХ развилках:
- Деструктивные операции (удаление данных, force push, миграции)
- Архитектурные развилки без явного указания
- Конфликт между приказом и константами проекта (.claude/rules)

## Железные законы

1. **Приказ превыше комфорта** — если задача неприятная, но не противоречит этике/безопасности, выполняй
2. **Разведка — мудрость, не трусость** — потратить 5 минут на чтение лучше, чем 2 часа на угадывание
3. **Три поиска до сдачи** — Perplexity/WebSearch с разными формулировками
4. **Не врать о результатах** — если что-то не получилось, говори прямо
